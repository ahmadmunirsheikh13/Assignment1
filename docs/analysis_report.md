# Job Analysis Report

## Overview

Analysis of 5 Data Science and Machine Learning job listings collected from major tech companies.

## Key Findings

### Top Skills
- Python: 5 occurrences
- Machine Learning: 4 occurrences
- SQL: 3 occurrences
- Statistics: 3 occurrences
- TensorFlow/PyTorch: 2 occurrences

### Top Locations
- California (Mountain View, Menlo Park, Los Gatos): 3 positions
- New York, NY: 1 position
- Seattle, WA: 1 position

### Top Companies
- Google: 1 position
- Meta: 1 position
- Amazon: 1 position
- Netflix: 1 position
- Spotify: 1 position

### Job Types
- Internships: 1 (Data Analyst Intern at Amazon)
- Junior Positions: 1 (Junior Data Scientist at Spotify)
- Senior Positions: 3 (Senior Data Scientist, ML Engineer, Data Science Manager)

### Most Common Job Titles
1. Senior Data Scientist (Google)
2. Machine Learning Engineer (Meta)
3. Data Analyst Intern (Amazon)
4. Data Science Manager (Netflix)
5. Junior Data Scientist (Spotify)

## Insights

The analysis reveals strong demand for Data Science skills in the tech industry, with Python being the most required technical skill. Major tech companies are actively hiring across different experience levels, from internships to senior management positions. California dominates as the primary location for these roles, though remote work options are increasingly common.

## Methodology

- Data collected from public job boards (Greenhouse, Lever)
- Analysis performed using pandas and matplotlib
- Skills extracted from job descriptions and requirements
- Location and company data aggregated for trend analysis
